Model Context Protocol & Claude

MCP (Model Context Protocol) lets Claude work directly inside your Notion workspace instead of just talking about it. By connecting the two, Claude can create pages, update databases, and search your content natively, meaning you can ask Claude to build project templates, organize notes, or manage tasks right in Notion without copying and pasting between apps.

Perfect for anyone who wants their AI assistant to actually live where they work.

Here's how to set it up:

  • Create a Notion integration - Visit notion.so/integrations, click "New Integration", name it, and select permissions like "Read content" and "Update content"
  • Get your token - Copy the "Internal Integration Token" from your integration for authentication
  • Install the MCP server - Download Node.js, then grab the Notion MCP server from the official GitHub repository
  • Configure Claude Desktop - Open Claude Desktop settings, go to Developer, click "Edit Config" to open claude_desktop_config.json, and add your server configuration with the token
  • Connect and restart - Fully quit and relaunch Claude Desktop so it can detect the MCP server and establish connection
